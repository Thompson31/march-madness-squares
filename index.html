🏀 One-Time Setup

This app uses a free Firebase Realtime Database for live sync. Takes ~5 minutes:

  1. Go to console.firebase.google.com → Add project
  2. Click Realtime Database → Create database → any region → Start in test mode
  3. Project Settings (gear) → Your apps → click </> Web → register → copy the firebaseConfig
  4. Open index.html in a text editor → find FIREBASE_CONFIG at the top → paste your values
  5. Drag index.html to app.netlify.com/drop → instant URL!
